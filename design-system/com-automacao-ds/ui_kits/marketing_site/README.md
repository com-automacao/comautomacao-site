# Marketing Site — Com Automação

Mobile-first interactive prototype of the Com Automação marketing website.

## Components
- `Header.jsx` — sticky blurred header with logo + nav
- `Hero.jsx` — fullscreen hero with tagline + CTAs
- `Services.jsx` — 3-column (1-column on mobile) service cards
- `Process.jsx` — numbered steps, Apple-style
- `CaseStudy.jsx` — full-bleed dark section with quote
- `ContactForm.jsx` — lead capture form
- `Footer.jsx` — minimal footer with logo + links
- `Button.jsx`, `Pill.jsx` — primitives

## Demo
`index.html` shows a mobile viewport by default (with a toggle for desktop), demonstrating scroll-driven reveals, sticky header blur, form interaction, and section transitions.
